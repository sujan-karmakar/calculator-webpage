# Calculator Webpage

A simple, responsive, and functional calculator webpage built using HTML, CSS, and JavaScript.

## Features

-   **Basic Operations:** specific arithmetic operations: Addition, Subtraction, Multiplication, Division.
-   **Input Validation:** Prevents invalid equations such as consecutive operators or starting with an operator.
-   **User Interface:** Clean, glassmorphism-inspired design with a responsive layout.
-   **Keyboard Support:** Supports both mouse clicks and keyboard input for calculations.
-   **Clear & Delete:** Includes 'AC' (All Clear) and 'DEL' (Delete) functions for easy correction.

## Technologies Used

-   HTML
-   CSS
-   JavaScript

## How to Run

1.  Clone the repository.
2.  Open `index.html` in any modern web browser.
